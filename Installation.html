Flutter Project Setup Documentation

1. Prerequisites

Ensure you have the following tools installed on your system:

  • Flutter SDK: Download from flutter.dev
  • Android Studio: Install from Android Developer (for Android development)
  • Xcode: (macOS only) for iOS development. Available on the Apple Developer website.
Download Flutter SDK

2. Setting Up Flutter

After installing Flutter SDK, follow these steps to set up your environment:

  1. Extract the Flutter SDK to a location on your computer.
  2. Add Flutter to your PATH environment variable:
    3. export PATH="$PATH:`flutter_bin_location`/flutter/bin"
  3. Verify your installation by running the following command in the terminal:
    4. flutter doctor

    The flutter doctor command checks your environment and displays missing dependencies.

3. Project Setup and Running

Follow these steps to open and run the project in your IDE:

  1. Open the Flutter project in your IDE (e.g., Android Studio or VS Code).
  2. Run the following command in the terminal to fetch dependencies:
    3. flutter pub get
  3. Connect an emulator or physical device to run the app. Then execute:
    4. flutter run

    Your app should now start on the connected device or emulator.

Flutter Test Drive

4. Building for Production

To build a release version of your app, use these commands:

  • Android: flutter build apk --release
  • iOS: flutter build ios --release

For more details on release builds, refer to Flutter Deployment Docs.
